honeypot-camera
===============

Observation camera honeypot (proof-of-concept)

Features:
 * Watermark
 * Daylight Intensity
 * Auto Refresh
 
Dependencies:
 * Tornado Web
 * Python PIL

Usage:
```bash
python camera.py
```
